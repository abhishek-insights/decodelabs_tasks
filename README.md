This project focuses on analyzing end-to-end supply chain performance by transforming raw operational data into meaningful insights using Power Query, Data Modeling, and Advanced DAX. It helped me understand how analytics can support better decision-making in areas such as inventory management, logistics efficiency, and vendor performance.

🚀 Project Objective
To analyze sales data and convert raw information into actionable business insights using Excel.

🎯 Key Insights

📉 Revenue shows a declining trend over time.
📈 Instagram is the top-performing referral source.
🛒 High-performing products: Printer, Chair, Laptop.
⚠️ High cancellation and return rates observed.
💳 Payment methods are evenly distributed.

🛠️ Tools Used
- Microsoft Excel
- Pivot Tables
